# 🔧 Refactoring Lab (Java)

This project demonstrates common **refactoring techniques** in Java using simple and clear examples.

---

## 📚 Implemented Techniques

### 1. Rename

Improved variable and method names for better readability.

### 2. Extract Method

Split large methods into smaller, reusable methods.

### 3. Guard Clauses

Simplified nested conditionals using early returns.

### 4. Replace Conditional with Polymorphism

Replaced `if/else` logic with class hierarchy.

### 5. Strategy Pattern

Encapsulated behaviors into interchangeable strategies.
